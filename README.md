1)Úvod
- Tohle je webová aplikace pro správu skladových zásob, produktů, jejich naskladnění a výdeji.


2)Naprogramované funkce
- Přídání produktů do databáze a mazaní z něj
- Přidání jednotlivých kategorií
- Evidence příjmu a výdeji zboží
- Historie událostí
- Vyhledávání a filtrování
- Stránkování
- Uživatelské účty spravované pouze adminem


3) Programovací jazyky
- Převážně PHP spolu s HTML
- JS
- CSS
- MySQL

4) Instalace
- Po stažení je potřeba importovat sql soubor do databáze
- Nastavení připojení k databázi v souboru 'database.php'
- Spuštění projektu na lokálním serveru

5) Struktura
- hlavní stránky: index.php, seznam.php, historie.php, prijem.php, vydej.php
- podstránky: product.php, sprava_uzivatelu.php apod.
- zbytek jsou jen funkční soubory

6) Databáze
Tabulky:
- uzivatele
- produkty
- kategorie
- historie
